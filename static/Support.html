Extreme Value Detection Techniques

The tool utilizes five advanced techniques to detect extreme values in the data:

1. Fixed Price Threshold (FPT)

The FPT method sets a threshold value, and any value beyond this limit is classified as a spike. This is one of the oldest techniques in time series analysis. The spike is represented as:

\[ Y_t^0 = \{Y_t : Y_t \geq a_1\} \]

2. Standard Deviation Filter on Prices (SFP)

The SFP method classifies a value as a spike if the absolute difference between the actual value and the mean value exceeds a specific standard deviation value. Typically, 3 standard deviations are used. The spike is represented as:

\[ Y_t^0 = \{Y_t : |Y_t - \overline{Y}| \geq 3\sigma\} \]

3. Moving Window Filter on Prices (MFP)

The MFP method applies a rolling window to the time series data and uses a similar approach as the SFP method. The spike is represented as:

\[ Y_t^o = \bigcup_{n=1,2,\ldots,N} \left\{ Y_{(\tau,\alpha)} : |Y_{(\tau_\alpha)}- \overline{Y} | \geq 1.96\sigma_n \right\} \]

4. Recursive Filter on Prices (RFP)

The RFP method repeats the standard deviation filtering procedure until no spikes remain.

5. Percentage Price Filter (PFP)

The PFP method identifies spikes based on quantile values, typically using the 2.5th and 97.5th percentiles. The spike is represented as:

\[ Y_t^0 = \{Y_t : Y_t \leq Y_t^{(2.5/100)} \} \cup \{Y_t : Y_t \geq Y_t^{(97.5/100)}\} \]

Extreme Value Replacement Techniques

Once extreme values are detected, they are replaced using one of the following four techniques:

1. Mean Value Replacement

This method replaces all detected spikes with the mean value of the entire dataset.

2. Median Value Replacement

This method replaces all detected spikes with the median value of the entire dataset.

3. Damping Scheme

This technique is less commonly used but is applied in this study. It uses a specific damping function to replace extreme values.

4. Threshold Scheme

This method replaces extreme values by taking the average of the mean and the median of the entire dataset.

Best Model Selection

For optimal forecasting, the best model was chosen based on the dataset treated with the Fixed Price Threshold (FPT) technique in combination with the Damping scheme. All the forecasts generated by this tool are based on this carefully selected model, ensuring the highest accuracy and reliability in predictions.
